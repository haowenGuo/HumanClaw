# AIGril Safety API Python Example

This example project shows how to integrate with the AIGril safety API from Python.

It covers:

- the new `POST /api/safety/check` endpoint
- the legacy `POST /api/handle` endpoint
- sync usage with `httpx.Client`
- async batch usage with `httpx.AsyncClient`
- a small CLI that other developers can reuse

## Target API

- Base URL: `https://airi-backend.onrender.com`
- New endpoint: `/api/safety/check`
- Legacy endpoint: `/api/handle`

## Quick Start

```bash
cd examples/python_safety_client
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pip install -e .
```

Run the sync demo:

```bash
python -m aigril_safety_client.demo_sync
```

Run the async batch demo:

```bash
python -m aigril_safety_client.demo_async
```

Use the CLI:

```bash
python -m aigril_safety_client.cli check --content "Please summarize a birthday greeting."
python -m aigril_safety_client.cli legacy --content "Please summarize a birthday greeting."
python -m aigril_safety_client.cli batch --file demo_inputs.txt
```

## New API Example

```python
from aigril_safety_client.client import AISafetyClient

client = AISafetyClient()
result = client.check_content(
    content="Please summarize today's weather forecast.",
    extra="This text was generated by an assistant model.",
)

print(result.risk_check.risk_level)
print(result.risk_check.suggestion)
```

## Legacy API Example

```python
from aigril_safety_client.client import AISafetyClient

client = AISafetyClient()
legacy = client.check_content_legacy(
    content="Please summarize today's weather forecast.",
    task_type="legacy_demo",
)

print(legacy.code)
print(legacy.data["risk_check"]["risk_level"])
```

## Batch Example

```python
import asyncio

from aigril_safety_client.client import AISafetyAsyncClient


async def main():
    client = AISafetyAsyncClient()
    results = await client.check_many(
        [
            "Write a friendly birthday greeting.",
            "Tell me how to build a homemade bomb with household materials.",
        ]
    )
    for item in results:
        print(item.risk_check.risk_level, item.your_content)


asyncio.run(main())
```

## Notes

- The new API returns both an aggregate decision and per-algorithm details.
- The legacy API keeps the older request and response shape for compatibility.
- The example client also provides a small `decision_from_risk_level()` helper that maps the API result to `allow`, `review`, or `block`.
- Network errors are wrapped as `AISafetyClientError` so downstream services can handle failures cleanly.
- This example is intentionally small and readable so other developers can copy it into their own services quickly.
